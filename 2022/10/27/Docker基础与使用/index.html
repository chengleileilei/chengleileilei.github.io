Docker基础与使用

命令

docker image ls 查看docker已安装镜像

docker ps -a查看当前运行docker镜像（不带-a表示查看所有docker镜像）

docker run -itd --name mycontainer --gpus all imagename将imagename镜像启动为mycontainer容器，并在后台运行

docker exec -it mycontainer /bin/bash 进入mycontainer容器

sudo docker commit -a "nathan" -m "create new img" eda05ad514f8 consul:v0 容器打包为镜像

docker tag 【镜像ID】【镜像名称】:【tag版本信息】 修改镜像name
